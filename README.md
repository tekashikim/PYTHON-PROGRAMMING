Description-This repository contains solutions for the Python Practical Assignment covering Python data types, control structures, functions, and object-oriented programming.

Files

1. integer.py– Demonstrates integer variables and arithmetic operations.
2. float.py– Demonstrates float variables and decimal calculations.
3. boolean.py– Demonstrates Boolean variables and comparison operators.
4. string.py– Demonstrates string operations, indexing, slicing, and methods.
5. list.py– Demonstrates creating, updating, removing, and looping through a list.
6. tuple.py– Demonstrates tuple creation and accessing tuple elements.
7. set.py– Demonstrates set operations and automatic removal of duplicate values.
8. dictionary.py– Demonstrates creating, accessing, and updating a dictionary.
9. selection.py– Demonstrates if, if...else, and if...elif...else statements.
10. looping.py– Demonstrates for loops and while loops.
11. builtin_functions.py– Demonstrates the use of Python built-in functions.
12. user_defined_functions.py– Demonstrates user-defined functions.
13. lambda_functions.py– Demonstrates lambda functions with sorted(), filter(), and map().
14. classes.py– Demonstrates classes and objects in Python.
15. class_variables.py– Demonstrates the use of class variables.
16. class_methods.py– Demonstrates the use of constructors, class variables, and class methods.

How to Run

1. Make sure Python is installed.
2. Open a terminal in this project folder.
3. Run any file using:

```bash
python filename.py
```

Example:

